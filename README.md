# AdventOfCode Solver

## Installation

You will need you AdventOfCode session ID to use this project.

```sh
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
echo 'AOC_SESSION_ID="your_own_session_id"' > .env
```

## Usage

```sh
./app.py --help
./app.py --year 2022 --day 1
```
